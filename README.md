MP3 Tag Reader and Editor

A command-line based application written in C to read, display, and edit ID3v2 metadata of MP3 files. This project allows users to view and modify MP3 tag information without affecting the actual audio content.

Features

->Read MP3 ID3v2 tags

->Display metadata such as:

  =>Title

  =>Artist

  =>Album

  =>Year

  =>Genre

  =>Comments

->Edit selected tag fields

->Works directly on MP3 binary files

->Menu-driven command-line interface

Technologies Used

->C Programming Language

->File Handling

->Structures and Pointers

->String Manipulation

->Binary File Processing
